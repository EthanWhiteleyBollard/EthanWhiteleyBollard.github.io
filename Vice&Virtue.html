Vice & Virtue

PROJECT TYPE - Personal/University

SOFTWARE USED - Unreal Engine 5, Blender

LANGUAGES USED - Blueprints

Vice & Virtue is a combination of different projects I have worked on through out University and in my own time. It is a personal project of mine that I've wanted to make for a while. Within Vice & Virtue you are tasked with solving different case in a rougelite manner. Each case is randomly generated so they don't feel the same as the last one.
