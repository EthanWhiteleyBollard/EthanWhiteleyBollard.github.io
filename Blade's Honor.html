Blade's Honor

PROJECT TYPE - University

SOFTWARE USED - Unity

LANGUAGES USED - C#

Blade's Honor is a project I worked on for a University module. This game is a 2 player fighting game. It is meant to emulate the combat of the game For Honor. Each player has 3 stances they can fight in with multiple different attacks and defence patterns.
