SkyHook

PROJECT TYPE - University

SOFTWARE USED - Unreal Engine 5

LANGUAGES USED - Blueprints

This was a collaborative game created with a team of other students for a module. We chose to create a game that feels fun and quick to play. So creating Skyhook we aimed for a grapple speedrunning game that feels smooth and fun. Throughout this project I worked on a variety of different things but primarly I was tasked with making a lot of the UI systems within the game.
